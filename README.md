# Blog Platform - Cypress BDD Tests

**Eğitmenler:** Nurettin Şenyer & Ömer Durmuş

## Proje Hakkında

Rails API backend ile çalışan blog platformu. Cypress ve Cucumber kullanılarak BDD testleri geliştirilmiştir.

## Teknolojiler

- HTML, CSS, JavaScript
- Cypress + Cucumber (BDD)
- Postman API Tests

## Çalıştırma

**HTTP Server:**

```bash
npx http-server . -p 8080
```

**Cypress Tests:**

```bash
npx cypress open
```

**Video Kaydı:**

```bash
npx cypress run
```

## Modeller

1. User - Kullanıcı kaydı
2. Category - Kategoriler
3. Post - Blog yazıları
4. Comment - Yorumlar
5. Like - Beğeniler
